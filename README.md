# AI Resume Analyzer

AI-powered Resume Analyzer built using Python and Streamlit.

## Features
- Upload Resume PDF
- Resume Analysis
- Skill Extraction
- Resume Score
- Improvement Suggestions

## Tech Stack
- Python
- Streamlit
- PDF Processing
- NLP

## How to Run

1. Clone the repository
2. Install dependencies

```bash
pip install -r requirements.txt
streamlit run AI_Resume_Analyzer/app.py
```
